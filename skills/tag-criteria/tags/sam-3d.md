# SAM 3D Tag Definition

### #sam-3d
- **Target**: SAM 3D (Segment Anything Model 3D) related content
- **Prerequisite**: Only apply to notes that already have `#3d-reconstruction` tag
- **Criteria**:
  - Keywords: SAM 3D, SAM3D, Segment Anything 3D, Meta 3D
  - Single image to 3D reconstruction/generation
  - Combines segmentation with 3D generation
  - References to Meta's 3D reconstruction research
  - Feed-forward 3D generation from single RGB image
- **Exclusions**:
  - SAM (Segment Anything Model) without 3D context -> do not apply
